🤖 HH.ru AI Recruitment Automation (ATS)

Система автоматизации подбора персонала, интегрированная с HeadHunter API и Google Gemini AI. Скрипт предназначен для автоматического сбора откликов, парсинга резюме и интеллектуального анализа кандидатов на соответствие вакансии.
🚀 Основные функции

    Синхронизация с HH.ru: Автоматическое получение списка активных вакансий и новых откликов.

    Парсинг резюме: Глубокий сбор данных о кандидате, включая опыт работы, навыки и зарплатные ожидания.

    AI-Аналитика (Gemini): Использование нейросети для сравнения требований вакансии с реальным опытом кандидата.

    Скоринг: Автоматическое вычисление процента соответствия (perc_match) и составление краткого резюме (summary) по каждому кандидату.

    Google Sheets Integration: Полная работа в интерфейсе Google Таблиц — создание отдельных листов под каждую вакансию и динамическое обновление данных.

🛠 Технологический стек

    Language: Google Apps Script (JavaScript)

    AI: Gemini Pro API

    API: HH.ru API (OAuth 2.0)

    Storage: Google Spreadsheet

📋 Структура проекта

    main.js — основная логика управления процессами.

    hh_module.js — взаимодействие с API HeadHunter (авторизация, получение вакансий/резюме).

    ai_assistent.js — логика взаимодействия с Gemini API и промпты.

    gs_module.js — вспомогательные функции для работы с Google Таблицами (запись, чтение, создание листов).

    appsscript.json — манифест проекта с настройками доступов.

⚙️ Настройка
1. Подготовка Google Таблицы

Создайте лист с названием prompts и следующими заголовками в первой строке:
vacancy_id, position, skills, perc, status_api.
2. Настройка свойств скрипта

В интерфейсе Google Apps Script перейдите в Project Settings и добавьте следующие Script Properties:

    client_id: ID вашего приложения в HH.ru

    client_secret: Секретный ключ вашего приложения HH.ru

    redirect_uri: Ваш Redirect URI (обычно URL скрипта)

    access_token / refresh_token: Токены доступа HH.ru

3. Настройка AI

Убедитесь, что функция callGeminiApi (в файле ai_assistent.js) содержит ваш рабочий API-ключ Google AI Studio.
🖥 Использование

    getVacIds: Запустите для импорта ваших активных вакансий из HH в таблицу.

    getResume: Скрипт соберет все новые отклики и создаст для каждой вакансии отдельный лист.

    aiAnalyst: Запускает процесс оценки кандидатов нейросетью. Скрипт проанализирует опыт и выставит оценку соответствия.

💡 Особенности реализации

    Контроль лимитов: В скрипт встроены паузы (Utilities.sleep), чтобы не превышать квоты API HH.ru и Google.

    Force Push: Для отправки изменений из Google Apps Script в этот репозиторий используется clasp.
